# 📊 Global Data Analytics Job Market Analysis

## 📌 Project Overview

This project explores the global data analytics job market, focusing on job roles, salary trends, degree requirements, and how salaries vary across different countries. The goal is to provide insights into the demand and compensation of data analytics professionals worldwide.

---

## 🎯 Objectives

* Analyze common job titles in the data analytics field
* Examine salary distribution across different roles
* Compare salaries across various countries
* Identify degree requirements for data-related roles
* Highlight key factors influencing salary differences

---

## 📁 Dataset

The dataset used in this project includes information on:

* Job titles in data analytics
* Salaries (entry-level to experienced)
* Country-specific salary data
* Degree requirements for each role

---

## 🛠 Tools & Technologies

* Python (pandas, numpy, matplotlib)
* Power BI (for dashboard creation and visualization)

---

## 📊 Key Features of the Dashboard

* Salary comparison by job title
* Salary distribution across countries
* Degree requirements analysis
* Interactive visualizations for better insights

---

## 💡 Key Insights

* Salaries vary significantly depending on the country
* Developed countries tend to offer higher salaries
* Technical skills (e.g., Python, SQL, Power BI) increase earning potential
* Most roles require at least a bachelor's degree
* Data analytics is a high-demand career globally
